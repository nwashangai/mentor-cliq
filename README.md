# MentorcliQ
## Requirements
- nodejs v12 or later


## Installation
From your terminal

1. clone the repository `git clone git@github.com:nwashangai/mentor-cliq.git`
2. `cd mentor-cliq`
3. run `npm install` 
4. run `npm run dev` 

open `http://localhost:3000` on browser

<i>Note:</i> This is currently only verified on Linux environment
